
## Description

This a personal API, build for personal usage.

## Installation
copy env and add your own OPENAI key to have application be fully operational
```bash
$ cp .env.example .env

$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
